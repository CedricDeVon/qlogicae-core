<h2 id="about">
  📖 About
</h2>
<h3 id="about-description">
  🧾 Description
</h3>
<p>
  <strong>QLogicae Core</strong> is a Windows x64 C++ framework with the aim to reuse code across integrated QLogicae projects.
  
  As of now, Desktop and CLI applications can be reliably integrated, however, Mobile and Web-based application integrations are not off-the-table. We are just prioritizing the first two.

</p>
<h3 id="about-core-features">
  ⚙️ Core Features
</h3>
<ul>
  <li>
    <h3>Built-in QLogicae API</h3>    
  </li>
  <li>
    <h3>Wrappers of Trusted Vcpkg C++ Packages</h3>    
  </li>
</ul>

</br>


<h2 id="getting-started">
  🚀 Getting Started
</h2>
<h3 id="getting-started-pre-requisites">
  📋 Pre-requisites
</h3>
<p>
  Please meet the requirements below to ensure a smooth development experience:
</p>
<ul>
  <li><p>OS - Windows 10/11</p></li>
  <li><p>System Architecture - x64</p></li>
  <li><p>Vcpkg - 2025-05-19 (Or Higher)</p></li>
  <li><p>Visual Studio - 3.14 (Or Higher)</p></li>
</ul>
